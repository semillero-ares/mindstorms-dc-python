Control de LEGO Mindstorms desde Python

Aquí presentamos un tutorial para controlar desde Python dos versiones muy populares de LEGO Mindstorms el NXT y el EV3. Para cada uno de esto existen unos paquetes en python que nos permitirán el control por USB, Bluetooth o Wi-Fi (disponible solo en el EV3).

  • Para el LEGO Mindstorms NXT tenemos el paquete nxt-python
  • Para el LEGO Mindstorms EV3 tenemos el paquete ev3-dc

En ambos casos tambien necesitaremos dos paquetes auxiliares, los cuales son:

  • PyUSB para la conexión por cable USB.
  • PyBluez para la conexión Bluetooth.

Dependiendo de la conexión que queramos hacer, necesitaremos configurar nuestro computador para controlar exitosamente nuestro LEGO.

Requerimientos

Para controlar del LEGO con un computador necesitaremos:

  • Un LEGO MindStorms NXT o EV3
  • Un computador con Wi-Fi y/o Bluetooth dependiendo de la conexión. Si no tenemos un computador con Wi-Fi o bluetooth podemos comprar un dongle USB con esta capacidad.
  • Python instalado en el PC, la versión de Python debe ser 3.x hasta la versión 3.11. Si tenemos una versión superior a la 3.11, deberemos configurar un ambiente virtual.
  • Instalación de los paquetes necesarios.
